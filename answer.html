1. What is DOM and what is the purpose of DOM?

answer: Document Object Model. The interaction is done with a programming language or JavaScript. Now since DOM is an object representation, we have properties, methods like any other JavaScript object. With the help of which we can access everything on our page, modify it easily if needed.

2. How will you select HTML elements using DOM? Name the DOM methods.

Here, since DOM adds methods to work with our pages, we can easily do something on an HTML page:

  • Creating a new element
  • Remove any previous element
  • Modify any previous element
  • Modify, add, remove attributes
  • Add, remove, modify styling
  • Create events on the page to perform a specific action on the page when something like this happens
  • Creating events on a specific element, when something happens on this element we want to take such an action.
